---
description: Initialize a new Playwright test repository with these skills
---

1. Copy the `skills/` directory from this repository into your target test repository's `.claude/skills/` folder.
2. Copy `.env.example` to `.env` in the target repository and fill in the required variables (`BASE_URL`, etc.).
3. Run the following command in the target repository:
   ```bash
   Set up Playwright for testing
   ```
4. Follow the prompts to scaffold the directory structure and install dependencies.
5. Verify the setup by checking for the `playwright.config.ts` file and `tests/` directory.
